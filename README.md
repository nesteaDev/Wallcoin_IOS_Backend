# Wallcoin API

1. Clonar el archivo **.env.template** y renombrarlo a **.env**
2. Configurar las variables de entorno
3. Inicializar Contenedor docker
   ```
   $ docker-compose up -d
   ```
4. Iniciar proyecto en mode desarrollo
    ```
    $ npm run start:dev
    ```
